# FG Lab

FG Lab is a fighting game team-building and synergy platform.

Unlike traditional wikis, FG Lab focuses on helping players answer questions such as:

* Who should I pair with this character?
* What team fits my playstyle?
* Which routes should I learn first?
* What strengths and weaknesses does my team have?

Current Games:

* Dragon Ball FighterZ (Active)
* 2XKO (In Development)

Planned Features:

* Team Builder
* Synergy Explorer
* Route Discovery
* Matchup Notes
* Community Tech
* Multi-Game Support

Tech Stack:

* HTML
* CSS
* JavaScript
* GitHub
* Cloudflare Pages

Status:
FG Lab is currently in active development.

Feedback, ideas, and contributions are welcome.

## Internal 2XKO Research Vault

The direct `/games/2xko/research-vault` route collects source-backed notes before they become recommendations. It supports local capture, search, source/status/tag filters, browser persistence, and JSON export.

It is intentionally absent from public 2XKO navigation. Static hosting means the route is hidden, not access-controlled.
